# GetWarp

Lekki plugin na warpy w GUI dla wersji 1.21.

### Komendy
* /warp - Otwiera menu GUI.
* /warp create <nazwa> <blok> <slot> <permisja> - Tworzy nowy warp.
* /warp delete <nazwa> - Usuwa istniejący warp.

### Uprawnienia
* getwarp.admin - Zarządzanie pluginem (tworzenie/usuwanie).
* getwarp.<permisja> - Dostęp do konkretnego warpu (np. getwarp.vip). Permisja jest definiowana podczas tworzenia warpu.

### Cechy
* Złote i pogrubione nazwy warpów w menu.
* Przejrzyste GUI z dekoracyjnym obramowaniem z szyb.
* Pełna konfiguracja wiadomości w config.yml.
* Zapis danych w pliku warpy.yml.
